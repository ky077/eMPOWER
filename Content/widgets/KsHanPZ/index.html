屬性 類型 預設 描述 測試
typeSet string 'HanPZ'
設定畫面視覺上是否顯示該文字。
注意：仍會保留其空間位置，且 html 內依然會有該文字。
樣板預設皆有Han(漢字)、P(拼音)、Z(注音)的空間位置。
若不保留其空間位置，請搭配 hiddenSet 使用。
若 html 不要顯示該文字，請將該文字拿掉。eeee
html 亦可將該 <div>拿掉，但仍需搭配 hiddenSet 使用。
typePosition string 'left' 設定段落水平靠左、置中、靠右排列
hiddenSet string null 隱藏Han(漢字)、P(拼音)、Z(注音)的空間位置。
若 html 不要顯示該文字，請將該文字拿掉。
itemMarginTop number | string 0 設定文字margin-top，預設數值單位為px。
itemMarginRight number | string 0 設定文字margin-right，預設數值單位為px。
itemMarginBottom number | string 0 設定文字margin-bottom，預設數值單位為px。
itemMarginLeft number | string 0 設定文字margin-left，預設數值單位為px。
hFontSize number | string 48 設定Han(漢字)font-size，預設數值單位為px。
P(拼音)、Z(注音)為漢字的0.25em。
zt5Symbol string '˙' 設定Z(注音)輕聲符號。
